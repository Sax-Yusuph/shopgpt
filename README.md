## Shop ai (WIP)

Power your shopify sites with ai.

![Demo](media/full.png)

## Installation

1. cd browser-extension
2. pnpm i
3. pnpm build chrome (this will create a dist/chrome folder)
4. load the chrome folder into the browser.

Server:

1. cd server
2. pnpm i
3. pnpm dev
